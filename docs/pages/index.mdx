# Revind

**Revind** is an Open source Tailwind UI components to bootstrap your new apps, projects or landing websites. revind is framework agnostic, meaning it does not depend on any framework. It is build in mind to work on most front-end frameworks out there!

What does it mean? **re = react, v= vue, ind = tailwind**
